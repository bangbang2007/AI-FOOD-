# AI-FOOD-
### 1. 서비스명

AI 기반 냉장고 식재료 추천 서비스

### 2. 서비스 기능

사용자가 냉장고에 있는 식재료를 입력하면 AI가 이를 분석하여 만들 수 있는 요리를 추천하고 간단한 조리 방법을 제공하는 서비스이다.

### 3. 서비스 구조

식재료 입력 → AI 식재료 분석 → 요리 추천 → 결과 출력

---

### 4. 기능 분해

#### 기능 1. 식재료 입력

**세부 기능**

* 식재료 직접 입력
* 식재료 추가 및 삭제
* 입력 내용 저장

**구현 과정**

* 사용자가 보유 식재료 입력
* 입력 데이터 수집
* 식재료 목록 확인
* 분석 대기 상태로 저장

---

#### 기능 2. AI 식재료 분석

**세부 기능**

* 식재료 분류
* 사용 가능한 재료 확인
* 조합 가능한 요리 탐색

**구현 과정**

* 입력된 식재료 불러오기
* 식재료 데이터 정리
* AI 모델에 분석 요청
* 가능한 요리 목록 생성

---

#### 기능 3. 요리 추천

**세부 기능**

* 추천 요리 제공
* 필요한 추가 재료 안내
* 간단한 조리 방법 제공

**구현 과정**

* 분석 결과 불러오기
* 적합한 요리 선정
* 조리 정보 생성
* 추천 결과 저장

---

#### 기능 4. 결과 출력

**세부 기능**

* 추천 요리 표시
* 조리 방법 표시
* 추가 재료 표시
* 결과 저장

**구현 과정**

* 추천 결과 불러오기
* 화면에 결과 출력
* 사용자 확인
* 결과 저장

---

### 5. 사용자 이용 흐름

1. 식재료 입력
2. 추천 요청
3. AI 식재료 분석
4. 요리 추천 생성
5. 결과 출력
6. 조리 방법 확인

---

### 6. 기대 효과

* 냉장고에 있는 재료를 효율적으로 활용할 수 있다.
* 불필요한 식재료 구매를 줄일 수 있다.
* 음식물 쓰레기를 감소시킬 수 있다.
* 요리 메뉴를 쉽게 결정할 수 있다.
